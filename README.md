TO INSTALL THE PACKAGE INTO JULIA:
1. GO TO REPL
2. TYPE IN 'using Pkg' and press enter
3. TYPE IN 'Pkg.add(url="https://github.com/gianeering/Riemann")' and press enter
4. WAIT FOR IT TO DOWNLOAD
5. DONE, ENJOY!
